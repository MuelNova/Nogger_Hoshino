# Nogger_Hoshino
modules for own hoshino bot

### Parameters:
`「para」` 必须参数

` [para] ` 可选参数

## ASF_Plus
**Basic Usage**:`@Bot ASF [功能] [Para1] [Para2] ...`

- [ ] 『绑定』: `绑定 「Bot」`绑定一个bot到你的qq号
- [ ] 『查询』：`查询 「Bot」`查询一个bot的状态
- [ ] 『删除』：`删除 「Bot」`解绑一个bot
- [X] 『开启』：`开 「Bot」 [令牌]`开启一个bot
- [X] 『关闭』：`关 「Bot」`关闭一个bot
